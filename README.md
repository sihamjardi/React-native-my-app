# Afficher et styliser des images dans une application React Native

## Objectif
Afficher un logo local dans l’écran d’accueil (Welcome Screen) en utilisant le composant Image, puis appliquer un style (taille, bord arrondi) et une mise en page simple, avec une palette de couleurs imposée.

---

## Pré-requis
Projet React Native déjà créé (Expo ou React Native CLI)
Un écran WelcomeScreen existe déjà (même minimal)
Un fichier image PNG pour le logo (ex: logo.png)
